About HTML Elements

HTML (HyperText Markup Language) consists of different elements that help structure web pages.

Common HTML Elements

  • <h1> to <h6> - Heading tags
  • <p> - Paragraph tag
  • <a> - Anchor tag for links
  • <ul> and <ol> - Lists
  • <div> - Container for content
